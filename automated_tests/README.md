## 📱 微信开发者工具操作指南

### 自动启动方式

运行 `start_full_demo.bat` 或 `wechat_demo.bat` 后，脚本会：
1. 🔍 自动查找微信开发者工具
2. 🚀 自动启动并导入项目
3. 📋 提供详细的页面演示指南

### 手动操作方式（如果自动启动失败）

1. **打开微信开发者工具**
   - 在电脑上找到并启动微信开发者工具

2. **导入项目**
   - 点击"+"号或"导入项目"
   - 项目目录选择：`health/miniprogram/`
   - AppID可以选择"测试号"

3. **编译运行**
   - 点击顶部工具栏的"编译"按钮
   - 等待编译完成，查看左侧模拟器
   - 确保没有红色错误提示

4. **开始演示**
   - 在模拟器中点击各个功能
   - 按照下面的页面顺序进行演示

## ⚙️ 配置微信开发者工具

如果自动启动失败，可以手动配置：

编辑 `automated_test_suite.py`，在 `WECHAT_DEV_TOOL_PATHS` 中添加你的路径：

```python
WECHAT_DEV_TOOL_PATHS = [
    "C:\\你的\\路径\\微信web开发者工具\\cli.bat",
    "C:\\Program Files\\Tencent\\微信web开发者工具\\cli.bat",
    # 添加更多路径...
]
```

或者手动打开微信开发者工具，导入项目目录：
```
项目路径：health/miniprogram/
```
