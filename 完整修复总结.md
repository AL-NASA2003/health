## 📋 用户需要手动操作的步骤

### 步骤 1：配置微信开发者工具（必须！）

1. **打开微信开发者工具**
2. **点击右上角"详情"按钮**
3. **切换到"本地设置"选项卡**
4. **勾选这个选项（非常重要）：**
   ```
   ✅ 不校验合法域名、web-view（业务域名）、TLS 版本以及 HTTPS 证书
   ```
5. **如果已经勾选，先取消，再重新勾选**

### 步骤 2：清除小程序缓存

1. 在微信开发者工具中，按 `Ctrl + Shift + Delete`
2. 选择 **"清除全部缓存"**
3. 点击 **"清除"** 按钮

### 步骤 3：完全重启微信开发者工具

1. **完全关闭微信开发者工具**（所有窗口都要关）
2. 等待 5-10 秒
3. **重新打开项目**

### 步骤 4：重新编译

1. 在微信开发者工具中，按 `Ctrl + B` 重新编译
2. 或者点击"编译"按钮

### 步骤 5：验证修复

**检查 Console 日志：**
应该看到：
```
=== 请求信息 ===
BASE_URL: http://localhost:5000/api
请求地址: http://localhost:5000/api/...
================
```

**检查 Network 面板：**
- Request URL 应该是 `http://localhost:5000/api/...`
- 不应该是 `https://localhost:5000/api/...`
- Status 应该是 `200 OK`
