## 🚀 使用方法

### 1. 生成虚拟数据

运行虚拟数据生成脚本，将测试数据导入数据库：

```bash
cd test_cases
python generate_test_data.py
```

这将生成：
- 32 个测试用户
- 128 条饮食记录（每个用户4条）
- 64 条运动记录（每个用户2条）
- 96 条饮水记录（每个用户3条）
- 20 个论坛帖子
- 64 条手账记录（每个用户2条）
- 40 条评论

### 2. 运行测试用例

使用 pytest 运行所有测试：

```bash
# 运行所有测试
pytest test_cases/ -v

# 运行特定文件的测试
pytest test_cases/test_001_user_login.py -v

# 运行特定测试用例
pytest test_cases/test_001_user_login.py::TestUserLogin::test_login_success -v

# 生成测试报告
pytest test_cases/ -v --html=test_report.html
```

## 📝 测试用例编写规范

每个测试用例都包含：
- **编号**: 唯一的测试用例编号 (001-128)
- **功能描述**: 清楚说明测试什么
- **输入**: 测试的输入数据
- **预期结果**: 期望的输出或行为
- **优先级**: 高/中/低

## 🔧 注意事项

1. **数据库**: 测试使用内存数据库 (SQLite in-memory)，不会影响生产数据
2. **API端点**: 测试中的API端点可能需要根据实际项目调整
3. **智谱AI**: AI相关测试需要有效的API Key
4. **网络**: 部分测试需要网络连接
